1. Compute Pay

The input consists of data for several workers. For each worker, the data consists of his or her name (a string that may include embedded blanks), title (a string that may include embedded blanks), wage (a floating-point value) and the number of hours he or she worked (an integer). The program must calculate each worker’s gross pay (wage times hours) and print it along with the worker’s name and title.

Sample Input:

Herb Green, Programmer, 12.5, 40

Sample Output:

Pay $480.00 to Herb Green, Programmer

2. Golf Scores

For this program, the input contains exactly 18 lines (one for each hole on a golf course). Each line contains 5 values: the first is the par for the hole; the remaining values are the scores (number of strokes) for each of four golfers (A, B, C, and D). Compute each golfer's score (which is the number of strokes they took minus par, summed over all holes). Print out all 4 golfer's scores and identify the winner.

3. More Vowels, Really?

Here, the input is a single string from the user (which could contain spaces). Read in this string, then print out the number of times each vowel appears in the string ('y' is not a vowel for our purposes today).

4. Scooped Out Square

The first number is the dimension of a square; the second number is the dimenion of a second 'hollow' square in the center of the first. Your program should draw the specified shape.
